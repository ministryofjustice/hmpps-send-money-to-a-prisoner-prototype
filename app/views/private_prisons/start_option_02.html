You can use this service to make a payment by Visa, Mastercard or Maestro debit card. Money usually takes less than 3 working days to reach a prisoner’s account, but it may take longer.

This service is free and secure, and available in all prisons in England and Wales.

Start now

Before you start

You’ll need the:

  • prisoner’s date of birth
  • prisoner number

Help using the service

You can get help with making a payment, ask a question about a payment you’ve already made, or ask something else about the service.

Alternatives to this service

If you can't use this digital service because you don't have a bank account, you can find out how to set up a bank account.

If you can't use this service because you don't have access to a computer or the internet, you can apply for an exemption which may allow you to send money through the post.
